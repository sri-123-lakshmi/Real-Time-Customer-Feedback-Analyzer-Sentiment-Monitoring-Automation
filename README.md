# Real-Time-Customer-Feedback-Analyzer-Sentiment-Monitoring-Automation
A Python-based customer feedback monitoring system that reads feedback from multiple sources, performs rule-based sentiment analysis, categorizes service issues, triggers alerts for negative feedback, and generates a daily summary report. Designed to simulate real-world customer service monitoring workflows.
